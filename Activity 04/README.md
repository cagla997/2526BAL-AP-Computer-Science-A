# ACTIVITY 04 - Finding All Prime Numbers in a Specific Range Program in JAVA
 As you know, prime numbers are positive integers that have no positive divisors other than themselves and one. The mathematician 
will give the program a lower and an upper limit. The program will then find all the prime numbers within these lower and upper limits 
and print them to the screen, leaving a space between each entry.
